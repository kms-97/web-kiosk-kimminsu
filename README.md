## Getting Started
### 1. 클론 레포지토리

```
https://github.com/woowa-techcamp-2022/web-kiosk-kimminsu.git
```

### 2. NPM 패키지 설치

```
npm install
```

### 3. 환경변수 설정
루트 디렉토리에 아래의 환경변수를 가진 `.env` 파일을 생성합니다.
```
DATABASE_TYPE=
DATABASE_HOST=
DATABASE_PORT=
DATABASE_USERNAME=
DATABASE=
REACT_APP_BASE_URL=
```

### 4. 클라이언트 빌드

```
npm run client:build
```

### 5. 서버 실행
기본 포트는 3000번입니다.
```
npm run server:start
```
